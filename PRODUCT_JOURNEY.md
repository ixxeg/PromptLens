# Local Image Metadata Catalog - Product Journey

Ниже - краткая история развития приложения от первого рабочего прототипа до релизной версии `3.0.0`.

## v0.0.1 - Первый рабочий прототип
- Сканирование одной папки с изображениями.
- Базовый каталог миниатюр.
- Отображение сырых метаданных выбранного изображения.
- Простая структура интерфейса: список слева, метаданные справа.

## v0.1.0 - Основа для реального использования
- Поддержка нескольких папок одновременно.
- Улучшенное извлечение генеративных метаданных (Prompt, Negative Prompt, Sampler, Steps, CFG, Seed, Model).
- Базовая очистка/нормализация значений из PNG/EXIF/JSON.

## v0.2.0 - Удобство просмотра
- Прокрутка каталога по вертикали и горизонтали.
- Регулировка размера миниатюр.
- Управление количеством колонок.
- Исправления ранних багов UI и некорректных пересчетов layout.

## v0.3.0 - Читаемая панель метаданных
- Перестроен вывод метаданных в структурный формат.
- Фокус на ключевых полях для генерации.
- Улучшенная типографика и визуальные акценты секций.

## v0.4.0 - Полноразмерный просмотр
- Открытие изображения по двойному клику.
- Zoom in/out, fit-to-window, reset zoom.
- Исправлено поведение колесика мыши для масштабирования.

## v1.0.0 - Визуальный редизайн (пастельная тема)
- Современная пастельно-фиолетовая тема приложения.
- Обновленные кнопки, элементы управления, аккуратные hover-эффекты.
- Мягкая визуальная полировка интерфейса без перегруза.

## v1.3.0 - Персонализация и фильтры
- Избранное для изображений.
- Пользовательские теги и сохранение тегов.
- Кнопка копирования позитивного промпта.
- Фильтрация, поиск и сортировка (`Newest/Oldest`).

## v1.6.0 - Стабильность сессий
- Сохранение состояния между запусками:
  - список выбранных папок,
  - UI-настройки,
  - favorites/tags.
- Автозагрузка и автоскан после старта.

## v2.0.0 - Производительность и масштаб
- Кэш миниатюр и кэш метаданных.
- Батч-рендер каталога для больших наборов.
- Оптимизация перерисовки при изменении размера миниатюр.
- Существенно более стабильная работа на больших папках.

## v2.3.0 - Точность парсинга генеративных данных
- Углубленный разбор ComfyUI/Workflow JSON.
- Улучшенное определение `Model`.
- Улучшенный парсинг LoRA, включая сложные и кастомные ноды.
- Фильтрация ложноположительных ассетов (VAE/CLIP/text encoder и т.д.).

## v2.6.0 - Архитектурная стабилизация каталога
- Перевод рендера миниатюр на Canvas-элементы.
- Устранены визуальные глитчи при быстрой прокрутке.
- Улучшена плавность и надежность отрисовки.

## v2.8.0 - Улучшение UX управления
- Удаление конкретной папки из списка сканирования.
- Стабилизация поведения правой панели при изменении размеров окна.
- Скрытие подписей миниатюр на маленьком размере для чистого вида.
- Кнопка показа/скрытия панели папок.

## v2.9.0 - Умный поиск и фоновая подготовка
- Поисковый индекс по ключевым данным изображения.
- Фоновый прогрев метаданных после скана.
- Debounce для сохранения состояния (меньше лишних операций записи).
- Более отзывчивый поиск и фильтрация в реальном использовании.

## v3.0.0 - Release: зрелый просмотрщик локальных генераций
- Полный рабочий цикл: сканирование, каталог, анализ метаданных, полноразмерный просмотр.
- Точная работа с Prompt/Negative/Model/LoRA и прочими параметрами генерации.
- Поддержка больших библиотек изображений с оптимизированной отрисовкой.
- Готовность к распространению через `.exe` сборку.
- Финальный UX-штрих: цветовые маркеры подпапок (`folder dot + tooltip`):
  - одна подпапка = один цвет,
  - файлы из корня выбранной папки - без точки,
  - современный контрастный стиль маркера в общей пастельной теме.
